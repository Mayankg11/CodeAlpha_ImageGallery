# Image Gallery

Nature's Gallery is a simple and responsive image gallery website built using HTML, CSS, and JavaScript.

The project displays beautiful nature images from different categories such as forests, rivers, mountains, and oceans. Users can filter images by category and view them in a fullscreen lightbox with previous and next navigation.

## Features

- Responsive image gallery layout
- Category-based image filtering
- Lightbox image preview
- Previous and next image navigation
- Smooth hover effects
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.

No additional setup or installation is required.
